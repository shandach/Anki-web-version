## Технические детали

### Tailwind CSS Breakpoints

```css
/* Mobile First подход */
базовые стили     /* 0px+ (mobile) */
sm:               /* 640px+ (tablet portrait) */
md:               /* 768px+ (tablet landscape) */
lg:               /* 1024px+ (desktop) */
xl:               /* 1280px+ (large desktop) */
2xl:              /* 1536px+ (extra large) */
```

### Ключевые адаптивные классы

**Отступы:**
- `px-3 sm:px-4 lg:px-8` - горизонтальные отступы
- `py-4 sm:py-6 lg:py-12` - вертикальные отступы

**Текст:**
- `text-lg sm:text-xl lg:text-2xl` - размеры заголовков
- `text-sm sm:text-base` - размеры обычного текста

**Сетки:**
- `grid-cols-1 sm:grid-cols-2 lg:grid-cols-3` - адаптивная сетка
- `flex-col sm:flex-row` - направление flex

**Размеры:**
- `h-14 sm:h-16` - высота навигации
- `min-h-[80px] sm:min-h-[100px]` - минимальная высота

**Интерактивность:**
- `active:scale-95` - анимация нажатия для touch
- `transition-transform` - плавная анимация

### Touch-Friendly элементы

Все интерактивные элементы имеют минимальную высоту 44px (рекомендация Apple/Google):
- Кнопки: `py-3` (12px × 2 + текст ≈ 44px+)
- Input поля: `py-2.5` на мобильных
- Навигационные ссылки: достаточный padding

### Предотвращение zoom на iOS

Input поля используют `text-base` (16px), что предотвращает автоматический zoom при фокусе на iOS Safari.
